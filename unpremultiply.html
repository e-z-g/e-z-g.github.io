About Unpremultiplication

How the algorithm works

Unpremultiplication reverses the premultiplied-alpha transform.

For each pixel where alpha > 0:
  R' = clamp(R / (A / 255), 0, 255)
  G' = clamp(G / (A / 255), 0, 255)
  B' = clamp(B / (A / 255), 0, 255)

Pixels with alpha = 0 are set to (0, 0, 0, 0).
Pixels with alpha = 255 are passed through unchanged.

All processing is done locally in your browser — no data is uploaded.
